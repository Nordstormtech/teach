Типы данных бывают изменяемые и неизменяемые (англ. mutable и immutable). 
К неизменяемым относятся целые числа (int), числа с плавающей запятой (float), булевы значения (bool), строки (str), кортежи (tuple). 
К изменяемым — списки (list), множества (set), байтовые массивы (byte arrays) и словари (dict).
